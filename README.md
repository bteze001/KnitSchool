# KnitSchool

A React + Vite learning app for knitting and crochet. It includes courses, lesson pages with step-by-step instructions, and different patterns and projects.

## Features 
- Course catalog with lesson pages
- Step-by-step lesson instructions
- Pattern library and stitch references
- Responsive UI with Tailwind

## Tech Stack
- React + TypeScript
- Vite
- React Router
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Getting Started 

### Prerequisites 
- Node.js 18+ (recommended)
- npm

### Install 
```bash
npm install
```

### Run locally 
```bash
npm run dev
```

### Build
```bash
npm run build
```

### Preview build 
```bash
npm run preview
```

## Project Structure
- `src/pages` - app pages (Home, Courses, Lesson, etc)
- `src/components` - shared UI components
- `src/data` - course and pattern data
- `public` - static assets
